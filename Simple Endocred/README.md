# Simple ENDOCRED

A minimal closed-economy model with **endogenous credibility**, used to introduce the
ENDOCRED mechanism. Open `.mod` with Dynare, `.jl` with Julia.

| File | Description |
| --- | --- |
| [simple_linear.mod](simple_linear.mod) | Linear endogenous-credibility model (Dynare) |
| [simple_linear.jl](simple_linear.jl) | Julia version of the linear model |
